1、注释、标识符、关键字、字面量

1.注释

单行注释：ctrl+/ 格式：//

多行注释：ctrl+shift+/ 格式：/* */

JavaDoc文档注释： 格式：/** */

2.标识符

java所有的组成部分都需要名字。类名、变量名以及方法名都被称为标识符。

命名规范

1.小驼峰命名法：(主要用于类)

规范一，标识符是一个单词的时候，所有字母小写 eg:name

规范二，标识符由多个单词组成的时候，从第二个单词开始，首字母大写 eg:firstName

2.大坨峰命名法：(主要用于类名)

规范一：标识符是由一个单词组成的时候，首字母大写 eg:Student

规范二：标识符是由多个单词组成的时候，每个首字母大写 eg:GoodStudent
